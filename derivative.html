derivatives

Derivatives tell you how a function is changing at any given point. They're like a speedometer for math‚Äîthey measure the rate of change or slope of a function. If you're looking at a graph, the derivative tells you how steep it is at a specific point. You'll use derivatives to find things like where a function is increasing or decreasing, where it‚Äôs flat, and where it curves up or down.

    Function Analysis

    The function analysis describes how derivatives behave and how they relate to the shape and behavior of a function. These are especially useful when analyzing graphs, motion, or solving application problems.

    Derivatives reveal important information about how a function increases, decreases, curves, and where it‚Äôs not smooth or not smooth.

    1. Linearity of Derivatives

    If f(x) and g(x) are both differentiable, and c is a constant:

    ddx [cf(x)g(x)] =c⋅f‚Ä≤(x)+g‚Ä≤(x)

  • You can take derivatives term-by-term, and pull constants out

    • 2. Differentiability Implies Continuity

    If a function is differentaible at a point, then it must also be continuous there.

    But:

    ‚ö†Ô∏è A function can be continuous without being differentiable (e.g., a sharp corner like |x| at x=0).

    3. Positive/Negative Derivatives and Monotonicity

    If f'(x) > 0 on an interval, f(x) is increasing there.

    If f'(x) < 0, f(x) is decreasing.

  • This connects derivatives to function behavior

    • 4. Second Derivative and Concavity

    If f''(x) > 0: the graph is concave up (shaped like a cup).

    If f''(x) < 0: the graph is concave down (shaped like a frown).

  • The second derivative helps analyze the curvature of the graph.

    • 5. Zero Derivative Doesn‚Äôt Always Mean a Maximum or Minimum

  • If f'(x) = 0, that point is critical, but it's not necessarily a max or min.
  • You need to check using first or second derivative test..

    • 6. Derivatives Can Fail to Exist

    A derivative does not exist at:

  • Sharp corners (like f(x) = |x| at x = 0)
  • Vertical tangents (like f(x) = ‚àõx)
  • Discontinuities (like a jump or hole)
