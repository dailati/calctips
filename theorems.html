theorems

Theorems in AP Calculus AB are like rules or facts that help you understand how functions behave. They explain important ideas like when a function has to be continuous, when it has a maximum or minimum, or when it must have a certain slope. These theorems don’t just tell you what happens—they tell you why it must happen, based on logic and math reasoning. You’ll use them to solve problems and justify your answers throughout the course.

Definition of Derivative

The definition of a derivative uses a limit to describe the instantaneous rate of change of a function at a point. It captures how a function's output changes as the input changes by a very tiny amount.

Rather than finding the average change over an interval, the derivative uses a limit to "zoom in" and find the exact slope at a single point.

Formal Definition

This version finds the general formula for the derivative of a function, which works for any value of x. It's your go-to when you want a rule like:

f′(x)= something involving x

It's used to build the derivative function, which can tell you the slope at any point on the curve.

This is the formula:

f′(x)= limh→0 f(x+h)-f(x) h

Alternative Definition

This version focuses on finding the derivative at one specific point x=a. It's like zooming in on the curve at just that location and asking:

"What's the slope right here, not everywhere?"

This is useful when you're only interested in behavior at a single value and don't need a whole derivative function.

This is the formula:

f′(a)= limx→a f(x)-f(a) x-a

Average Rate of Change

The average rate of change of a function over an interval measures how much the output value changes, on average, for each unit of change in the input.

In other words:
It's the slope of the secant line between two points on the graph.

This is the formula:

Average Rate of Change = f(b)-f(a) b-a

  • f(b)-f(a): change in output (Δy)
  • b-a: change in input (Δx)

It tells you how fast the function is changing on average between two points x=a and x=b——not at a specific point like the derivative.

Instantaneous Rate of Change

The instantaneous rate of change tells you how fast a function is changing at one specific point ——not over an interval.

Think of it like this: While the average rate of change is like speed over a trip, the instantaneous rate is your speed shown on the speedometer at one exact moment.

This is the formula:

Instantaneous Rate of Change at x=a is

f′(a)= limh→0 f(a+h)-f(a) h

So:

  • Instantaneous rate of change = derivative
  • It gives you the slope of the tangent line at one point on the graph

Squeeze Theorem

The Squeeze Theorem helps you find the limit of a function when it's hard (or impossible) to evaluate directly.

It works by "squeezing" the tricky function between two simpler ones that have the same limit at a certain point.

If:

g(x)≤f(x)≤h(x)

for all x near (but not necessarily equal to) a point a, and:

limx→ag(x) =limx→ah(x) =L

Then:

limx→af(x) =L

What It's Used For:

  • When direct substitution fails, like with sinxx as x→0
  • To handle weird or oscillating functions by trapping them between two known limits

Intermediate Value Theorem

The Intermediate Value Theorem (IVT) says that if a function is continuous on a closed interval [a,b], then the function must hit every y-value between f(a) and f(b) at least once.

If a function starts at one value and ends at another, and it doesn't "break" (i.e., it's continuous), then it must pass through every value in between.

If:

  • f is continuous on [a,b]
  • f(a) ≠ f(b)
  • N is any value between f(a) and f(b)

Then:

There exists a number c∈(a,b) such that f(c)=N

What It's Used For:

  • Proving a function hits a certain value
  • Showing that an equation has a solution (e.g. a root exists)

Mean Value Theorem

The Mean Value Theorem (MVT) connects the average rate of change of a function over an interval with its instantaneous rate of change at some point inside that interval.

If a function is continuous and smooth (differentiable), then somewhere in the interval , the function's slope (derivative) matches the average slope between the endpoints.

If:

  • f is continuous on [a,b]
  • N is differentiable on (a,b)

Then there exists some number c ∈ (a,b) such that:

f′(c)= f(b)-f(a) b-a

What It Means:

The slope of the secant line (average rate of change) equals the slope of the tangent line (derivative) at some point c.

It's like saying:

If you drove from point A to point B at an average of 60 mph, then at some moment, you were going exactly 60 mph.

What It's Used For:

  • Justifying that a function's slope must hit a certain value
  • Showing that a function's derivative matches a secant line's slope

Extreme Value Theorem

The Extreme Value Theorem (EVT) guarantees that a function will have a maximum and minimum value on a certain interval—as long as it's continuous.

If a function is continuous on a closed interval [a,b], then it must reach a highest and lowest point somewhere in that interval.

If:

  • f is continuous on the closed interval [a,b]
Then:
  • f has both an absolute maximum and an absolute minimum on [a,b]

What It Means:

  • The function must "level out" at its highest and lowest values somewhere.
  • These extrema can occur either:
    • At the endpoints a or b
    • Or at some interior point in (a,b)

What It's Used For:

  • Ensuring that real-world functions (like cost, height, or temperature) have a highest or lowest point
  • Key step in optimization problems
  • Supports the logic behind checking endpoints and critical points
